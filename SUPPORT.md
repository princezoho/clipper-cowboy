# Support

Clipper Cowboy is a community-maintained local-first beta with no support-level
guarantees.

- Use GitHub Issues for reproducible defects; include the requested environment
  and reproduction details.
- Use GitHub Discussions for questions or workflow ideas if Discussions is
  enabled for this repository.
- Report vulnerabilities privately as described in [SECURITY.md](./SECURITY.md),
  not in a public issue.

Never include API keys, `.env` contents, personal paths, or private media in a
support request.
